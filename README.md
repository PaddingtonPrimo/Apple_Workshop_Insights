# Apple Workshop Insights

A React application built with Vite.

## Live Site

This page is deployed to GitHub Pages at:

**https://paddingtonprimo.github.io/Apple_Workshop_Insights/**

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built output is placed in the `dist` directory and automatically deployed to GitHub Pages on every push to the `main` branch.
